# Disaster Relief Management System

A console-based Disaster Relief Management System developed in C that simulates disaster response operations using fundamental Data Structures and Algorithms.

## Features

- Relief request prioritization using a Priority Queue
- Camp management using a Binary Search Tree (BST)
- Delivery history tracking using a Stack
- Shortest path calculation using Dijkstra's Algorithm
- Menu-driven interface for easy interaction

## Data Structures & Algorithms Used

- Priority Queue
- Binary Search Tree (BST)
- Stack
- Graph
- Dijkstra's Algorithm

## Tech Stack

- C
- GCC

## How to Run

```bash
gcc disaster_relief.c -o disaster_relief
./disaster_relief
```

## Learning Outcomes

- Practical implementation of fundamental data structures
- Graph algorithms and shortest path computation
- Dynamic memory management in C
- Modular programming and menu-driven application design
